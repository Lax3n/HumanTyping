# Examples

This directory contains example scripts demonstrating how to use HumanTyping in various scenarios.

## Playwright Example

**File**: `playwright_example.py`

Demonstrates realistic typing in a Playwright automation script.

```bash
# Install Playwright
uv add playwright
uv run playwright install chromium

# Run the example
uv run examples/playwright_example.py
```

## Selenium Example

**File**: `selenium_example.py`

Shows how to integrate HumanTyping with Selenium WebDriver.

```bash
# Install Selenium
pip install selenium

# Run the example (requires ChromeDriver)
python examples/selenium_example.py
```
